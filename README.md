# Learning Journey

## Sobre o Projeto

O **Learning Journey** é um sistema de estudos estruturado dedicado à compreensão profunda e domínio completo de livros selecionados. Este projeto documenta o processo de aprendizado por meio de anotações manuscritas, utilizando exclusivamente métodos analógicos: papel, caneta e livros físicos.

Cada obra estudada recebe um plano de estudo personalizado, adaptado às suas características específicas, visando maximizar a compreensão, retenção e absorção conceitual do conteúdo.

## Objetivo

Desenvolver uma base sólida de conhecimento que integre o universo dos negócios com o domínio da tecnologia e da computação. O projeto busca construir uma visão multidisciplinar capaz de conectar estratégia empresarial, inovação tecnológica e fundamentos computacionais, formando competências essenciais para liderar na intersecção entre esses campos.

A maestria intelectual é alcançada através da disciplina da escrita manual, do pensamento crítico e da reflexão contínua. Este repositório serve como registro permanente de um compromisso de longo prazo com o aprendizado profundo e significativo.

## Componentes

### Pasta `#Prompts`
Contém modelos estruturados para:
- Criação de planos de estudo personalizados para cada livro
- Avaliação do progresso de aprendizado

### Pasta `Livros`
Organizada por obra estudada, onde cada livro possui:
- **Plano de estudo específico**: documento `.md` com metodologia personalizada
- **Pasta de PDFs**: armazenamento das anotações manuscritas digitalizadas

### Nomenclatura dos Arquivos
Os PDFs seguem o padrão: `[número]_[nome-do-livro]_[dd-mm-aaaa].pdf`

Exemplo: `1_meditacoes_02-02-2026.pdf`

## Metodologia

1. Seleção criteriosa de obras relevantes
2. Desenvolvimento de plano de estudo personalizado
3. Estudo sistemático com anotações manuscritas
4. Digitalização e arquivamento das anotações
5. Avaliação periódica do progresso
6. Ajustes metodológicos conforme necessário
